List Of Git Command

  1. git init: Initializes a new Git repository.
  2. git clone [repository_url]: Clones a repository into a new directory.
  3. git commit -m "[message]": Records changes in the staging area with a commit message.
  4. git add [file]: Adds changes in the working directory to the staging area.
  5. git status: Shows the status of changes as untracked, modified, or staged
  6. git diff: Displays changes between the working directory and the index or a commit
  7. git log: Displays the commit history.
  8. git branch: Lists all local branches.
  9. git branch [branch_name]: Creates a new branch.
  10. git checkout [branch_name]: Switches to the specified branch.
  11. git merge [branch_name]: Merges changes from one branch into the current branch.
  12. git remote: Lists remote repositories.
  13. git remote -v: Shows detailed information about remote repositories.
  14. >git fetch [remote_name]: Fetches changes from a remote repository.
  15. git pull [remote_name] [branch_name]: Fetches changes and merges them into the current branch.
  16. git push [remote_name] [branch_name]: Pushes changes to a remote repository.
  17. git branch -d [branch_name]: Deletes a branch.
  18. git stash: Stashes changes in the working directory.
  19. git stash apply: Applies the last stash to the working directory.
  20. git tag [tag_name]: Creates a new tag at the current commit.
  21. git remote add [remote_name] [remote_url]: Adds a new remote repository.
  22. git remote rm [remote_name]: Removes a remote repository.
  23. git reset [commit]: Resets the staging area to a specific commit.
  24. git revert [commit]: Creates a new commit that undoes changes made in a previous commit.
  25. git fetch --prune: Removes remote branches that no longer exist on the remote.
  26. git cherry-pick [commit_hash]: Applies changes from a specific commit to the current branch.
  27. git log --graph --oneline --all: Displays a concise, graphical representation of the commit history.
  28. git config --global user.name "[name]": Sets the global username for Git.
  29. git config --global user.email "[email]": Sets the global email for Git.
  30. git show [commit]: Shows the changes made in a specific commit.
  31. git remote show [remote_name]: Displays more information about a specific remote.
  32. git log --author="[author_name]": Filters the commit history by author.
  33. git blame [file]: Shows the commit history with line-by-line annotations.
  34. git log --since="[date]": Filters the commit history by date.
  35. git log --grep="[pattern]": Filters the commit history by a specific commit message pattern.
  36. git clean -n: Shows which files would be removed by git clean.
  37. git clean -f: Removes untracked files from the working directory.
  38. git remote update: Fetches updates from all remotes.
  39. git push --force: Forces the push of local changes to a remote repository.
  40. git log --oneline: Displays a condensed log with each commit on a single line.
  41. git log --stat: Shows statistics about modified files in each commit.
  42. git log --pretty=format:"%h - %an, %ar : %s": Customizes the log output format.
  43. git branch -m [old_branch_name] [new_branch_name]: Renames a local branch.
  44. git config --global core.editor "[editor]": Sets the default text editor for commit messages.
  45. git show-branch: Shows the branches and their commits.
  46. git rev-parse HEAD: Outputs the current commit hash.
  47. git bisect start: Initiates a binary search to find a specific commit.
  48. git reflog: Displays a reference log, showing the history of HEAD positions.
  49. git log --graph --decorate --oneline: Shows a compact graph of branch history.
  50. git rm [file]: Removes a file from both the working directory and the staging area.
